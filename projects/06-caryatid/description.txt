SPRING 2023. MICHAEL LANE BOLDT, JILLIAN LEEDY, GUARAV DUA, CASEY REHM. SOFTWARE: PIFUHD [EARLY HUMAN-FORM BASED IMAGE-TO-MESH GENERATION], HOUDINI, RHINO7+GRASSHOPPER, ROBOTSTUDIO, PROCESSING, COLAB, REALSENSE. WE EMPLOYED PHOTO MANIPULATION USING CUSTOM AI PIPELINES TO GENERATE HUMANOID CARYATID FORMS.RAPID ROBOT TOOLPATHS FOR THE 6-AXIS ABB IRB 6700 WERE GENERATED IN GRASSHOPPER TO CARVE A 0.5X0.5X1.0M STYROFOAM CARYATID WITH A STATIONARY HOT WIRE. THE FORM WAS THEN SPACKLED, SANDED TO CREATE A SMOOTH, UNIFORM FINISH. A RGB LIDAR SENSOR WAS USED TO CREATE A VISUAL INPUT OF THE ROBOT AND FED THROUGH A REAL-TIME AI PIPELINE - A FEEDBACK LOOP BETWEEN A PHYSICAL INPUT AND A CORRESPONDING STABLE DIFFUSION IMAGE. COLORS OF THE IMAGE WERE SORTED INTO 5 CHANNELS TO GENERATE COLOR-ISOLATED TOOLPATHS.
